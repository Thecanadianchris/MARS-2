# MARS Mk1 Original Reconstructed Backup

This ZIP was reconstructed from the source code pasted/uploaded in the ChatGPT conversation.

It is intended as a practical backup of the original Base44/Vite/React MARS Mk1 project files that were shared.

Important notes:
- This is not guaranteed to be byte-for-byte identical to the user's disk copy.
- Local assets/images were not included unless they were provided in the conversation.
- Base44 backend settings/entities/functions are not included because they live in the Base44 backend, not in the frontend source.
- The main MARS interface, showcase components, config, Base44 client, auth layout, and selected UI support files are included.

Key included areas:
- src/components/mars
- src/components/ui
- src/components/AuthLayout.jsx
- src/components/GoogleIcon.jsx
- src/api/base44Client.js
- src/lib/app-params.js
- Vite/Tailwind/PostCSS/JS config files
