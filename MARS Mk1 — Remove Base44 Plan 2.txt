MARS Mk1 — Remove Base44 Plan
Phase 0 — Safety

Keep two versions:

MARS-Mk1-Base44-Archive
MARS-Mk1-Standalone

Do not edit the archive. All work happens in the standalone copy.

Phase 1 — Make it run without Base44

Goal: app opens locally with no Base44 packages.

Remove:

@base44/sdk
@base44/vite-plugin
src/api/base44Client.js
src/lib/app-params.js

Change:

vite.config.js
package.json
App.jsx
ChatPanel.jsx
Login/Register/Forgot/Reset pages
AuthContext.jsx

Temporary approach:

Disable login.
Route / straight to Control.
Keep /showcase for the MARS website.
Store chat history in browser localStorage.

Result:

npm install
npm run dev

MARS opens and the UI works.
